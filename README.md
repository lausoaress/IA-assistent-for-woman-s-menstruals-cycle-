**AI Assistant for Women’s Menstrual Cycle**

An AI-powered assistant designed to help track and support women’s menstrual cycle management, providing organization, guidance, and insights using Artificial Intelligence (OpenAI).

**Features**

Menstrual cycle tracking support

Cycle-related organization and information

AI assistant integration (OpenAI / ChatGPT)

Backend server with Node.js


**Technologies Used**

Node.js

Express.js

OpenAI API

Dotenv


**1) Installation & Setup**

git clone https://github.com/YOUR-USERNAME/YOUR-REPOSITORY.git

**2) Enter the project folder**

cd IA-assistent-for-woman-s-menstruals-cicle--main/IA-assistent-for-woman-s-menstruals-cicle--main

**3) Install dependencies**

npm install 
(in powersheel/terminal vscode)

**in .env replace current key with your own OpenAI key**

**start**

node server.js

or, if the project includes a start script:

npm start
